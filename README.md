# Image Gallery Responsive

Image gallery with lazy loading, lightbox viewer, filtering by tags, and upload functionality.

## Features

- Responsive grid layout
- Lazy loading images
- Lightbox viewer
- Tag filtering
- Image upload
- Search functionality
- Masonry layout

## Tech Stack

- HTML5
- CSS3 (Grid/Flexbox)
- JavaScript
- Intersection Observer API

## Usage

Open `index.html` in your browser.
